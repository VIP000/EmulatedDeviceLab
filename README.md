# Emulated Device Lab (EDL)

Chrome extension that allows simultaneous testing on multiple emulated devices.

![EDL Preview](http://i.imgur.com/040qOyf.png)

产品特点

同步滚动，单击并在所有设备上的关键事件
同步样式的变化
提供单页，其中所有的设备可以被管理
用法

安装扩展
点击EDL动作按钮
添加设备
每个器件窗口上单击鼠标右键，选择“流EDL”
故障排除

之前报告错误，请阅读常见问题解答，并检查错误是不是已经报道。
